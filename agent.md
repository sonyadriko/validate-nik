# Agent Configuration

NIK Parser API - FastAPI service for parsing Indonesian NIK numbers.

## Agent Capabilities

- Parse 16-digit NIK into structured biodata
- Extract: gender, birth date, location (province/regency/district)
- Calculate: age, zodiac, Javanese pasaran
- Serverless-ready deployment (Vercel/AWS Lambda)

## Context

- Python 3.11+
- FastAPI + Pydantic
- Layered hexagonal architecture
- TDD with 80% coverage target
